# News Web App

An open-source web app using SSR provided by NextJs.

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`   

To Start Server:

`npm run dev`  

To Visit App:

`localhost:3000`
